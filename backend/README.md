# Backend Microservices

## What is this folder?
This folder holds the brain of our portal. It does all the hard thinking, saving data to the database, and sending messages. It's like the kitchen in a big restaurant where all the orders are cooked!

## Steps to use
1. Go into this folder using your terminal.
2. Run the code using Docker to build the images.
3. It will wait patiently for the frontend website to talk to it.

## Summary
This folder contains the backend microservices (like Forms Engine and API Gateway) that make the portal work.
