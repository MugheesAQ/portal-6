# Kubernetes Files

## What is this folder?
This folder holds instructions for Kubernetes. Kubernetes is like a traffic cop or a manager that makes sure all our little worker programs (microservices) are running smoothly and safely on the computers. 

## Steps to use
1. Open a terminal on your computer.
2. Type `kubectl apply -f .` to give the instructions to the traffic cop.
3. Watch your programs start running!

## Summary
This folder contains the rules for Kubernetes to run our citizen portal app.
